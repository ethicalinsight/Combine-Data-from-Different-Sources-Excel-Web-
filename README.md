# Combining Data from Multiple Sources

## Overview
This project integrates sales data from Excel for creating Sales Dashboard, and integrates external data from a web source to create a unified dataset for analysis to create a Real Time Dashboard.

## Features
- Data import from Excel files.
- Web data connection using Power BI (Get Data â†’ Web).
- Transformation and cleaning in Power Query Editor.
- Unified dashboard combining both sources.

## How to Use
1. Load the Excel dataset into Power BI.
2. Connect to the web source via Power BI.
3. Apply transformations in Power Query.
4. Visualize combined insights in the dashboard.
